# Brow App

Aplicativo Flutter completo para estética de sobrancelhas com Material Design 3.

## Estrutura do Projeto

```
lib/
├── main.dart              # Ponto de entrada da aplicação
├── routes/
│   ├── app_routes.dart    # Definição das rotas nomeadas
│   └── route_generator.dart # Gerador de rotas com animações
├── screens/
│   ├── home_screen.dart
│   ├── about_screen.dart
│   ├── services_screen.dart
│   └── contact_screen.dart
├── widgets/               # Widgets personalizados
├── models/                # Modelos de dados
└── services/              # Serviços e lógica de negócio
```

## Características

- ✅ Material Design 3 (Material You)
- ✅ Tema moderno com cores suaves
- ✅ Animações nativas (Fade + Slide)
- ✅ Rotas nomeadas
- ✅ Todas as telas com scroll
- ✅ Transições animadas entre telas

## Como Executar

1. Certifique-se de ter o Flutter instalado
2. Execute `flutter pub get` para instalar as dependências
3. Execute `flutter run` para iniciar o aplicativo

## Requisitos

- Flutter SDK >= 3.0.0
- Dart SDK >= 3.0.0
