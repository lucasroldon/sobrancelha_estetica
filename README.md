# Brow App

Aplicativo Flutter completo para estética de sobrancelhas desenvolvido pelo developer Lucas Roldon.

## Estrutura do Projeto

```
lib/
├── main.dart              # Ponto de entrada da aplicação
├── routes/
│   ├── app_routes.dart    # Definição das rotas nomeadas
│   └── route_generator.dart # Gerador de rotas com animações
├── screens/
│   ├── home_screen.dart
│   ├── about_screen.dart
│   ├── services_screen.dart
│   └── contact_screen.dart
├── widgets/               # Widgets personalizados
├── models/                # Modelos de dados
└── services/              # Serviços e lógica de negócio
```

App ainda em desenvolvimento.
